## 海风AT
是一款开源的策略开发平台.为用户提供方便易用的策略开发工具.

## 海风AT的功能
* 策略编写
    *  提供常用指标
    *  采用HLOC调用K线数据
* 历史数据
    *  提供数据日常维护服务
    *  提供实时数据分钟级服务
    *  提供分笔数据(内网)
